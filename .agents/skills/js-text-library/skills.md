# JavaScript Text Library

Use this skill when building animated, stylized, or interactive text effects in JavaScript.

## Recommended Libraries

### Anime.js

A powerful animation library that can target individual letters to make them float, spin, scale, or ripple like a cosmic wave.

### Baffle.js

A lightweight library for obfuscating and revealing text. It works well for scrambled sci-fi UI or matrix-style decryption effects.

### Blast.js

A library that separates text by character, word, or sentence, giving fine-grained control over styling and motion.

### TypeIt

A highly customizable typing animation library that can simulate an alien computer typing a message, including pauses and deletions.
