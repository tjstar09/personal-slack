# GitHub Repository URL Actions

You can use a GitHub repository URL to perform tasks, track progress, or fetch data. 
For all links below, replace `<git_repo_name>` with the owner and project name (e.g., `owner/project`).

## 📥 Downloading and Cloning

* **View Web Page:** `https://github.com<git_repo_name>`
* **Clone via HTTPS:** `https://github.com<git_repo_name>.git`
* **Clone via SSH:** `git@github.com:<git_repo_name>.git`
* **Download ZIP Archive:** `https://github.com<git_repo_name>/archive/refs/heads/main.zip`

## 🛠 Managing and Automating Work

* **View Actions Tab:** `https://github.com<git_repo_name>/actions`
* **Create New Workflow:** `https://github.com<git_repo_name>/actions/new`
* **Manage Runner Settings:** `https://github.com<git_repo_name>/settings/actions/runners`

## 📑 Viewing and Tracking Progress

* **View Issues List:** `https://github.com<git_repo_name>/issues`
* **Create New Issue:** `https://github.com<git_repo_name>/issues/new`
* **View Pull Requests:** `https://github.com<git_repo_name>/pulls`
* **Create Pull Request:** `https://github.com<git_repo_name>/compare`
* **View Commit History:** `https://github.com<git_repo_name>/commits`
* **View Project Releases:** `https://github.com<git_repo_name>/releases`
* **View Project Insights:** `https://github.com<git_repo_name>/pulse`

## 🤖 Connecting via APIs

* **Fetch Main API Data:** `https://github.com<git_repo_name>`
* **Fetch API Issues:** `https://github.com<git_repo_name>/issues`
* **Fetch API Pull Requests:** `https://github.com<git_repo_name>/pulls`
* **Fetch API Releases:** `https://github.com<git_repo_name>/releases`
* **Manage Webhook Settings:** `https://github.com<git_repo_name>/settings/hooks`
